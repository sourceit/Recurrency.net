WHAT IS RECURRENCY.NET
Recurrency.net is a recurring date library for .net.

E.g.  You can create calcualte

TEXT REPRESENTATION OF RECURRENCY

Type 1 char
Start date 8 chars D | W | M | Y
End date 8 chars (yyyymmdd, 00000000 if not specified)
Num recurrences 8 chars (00000000 if not specified)

Type specific info x chars

E.g 
D201112130000000000000012? will calculate the 12 days of xmas for 2011
